스케줄링 알고리즘 (비교 대상)

FCFS

선착순 처리

특징

  • 단순 FIFO 큐
  • 비선점형
  • O(1) 스케줄링

한계

호위 효과 발생, 차등 서비스 불가

Priority

우선순위 기반 + 에이징

특징

  • 4단계 우선순위 (URGENT~LOW)
  • 에이징으로 기아 방지
  • 동적 우선순위 조정

한계

에이징 없으면 기아 발생 가능

MLFQ

다단계 피드백 큐

특징

  • 적응형 분류
  • 큐별 타임 퀀텀
  • 비선점형 적응

한계

파라미터 튜닝 복잡

WFQ

가중치 공정 큐잉

특징

  • 가중치 비례 서비스
  • 가상 종료 시각 계산
  • JFI 공정성 지표

한계

VFT 계산 부담

보조 기능 (전처리 단계)

Rate Limiter

토큰 버킷 알고리즘

역할: 테넌트별 요청 빈도 제한

방법: 버스트 용량 제어

목적: 과부하 방지 (보조 수단)
