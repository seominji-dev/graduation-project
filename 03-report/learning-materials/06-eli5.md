# 06. "5살 아이에게 설명하기" (ELI5 - Explain Like I'm 5)

> **학습 목표:** 이 문서를 읽고 나면 복잡한 개념을 아주 단순하게 설명할 수 있습니다.

---

## Part 1: 프로젝트란 무엇인가?

### Q: 이 프로젝트이 뭐예요?

**A:** 

로봇 친구들에게 **똑똑하게 일하는 순서**를 정해주는 시스템이에요.

비유를 들어볼게요:

**놀이터에서의 미끄럼틀:**
- 친구들이 줄을 서서 기다려요
- 누가 먼저 탈까요?
  - 키 순서? 나이 순서? 온 순서?
  
**이 프로젝트:**
- AI(로봇)에게 일을 시키는 친구들이 많아요
- 누구를 먼저 도와줄까요?
- **똑똑한 순서**를 정해줘요!

---

## Part 2: 4가지 방법

### Q: 4가지 방법이 뭐예요?

**A:**

네 가지 다른 규칙이에요:

**방법 1: 선착순 (FCFS)**

```
놀이터 미끄럼틀:
1. 철수가 먼저 와서 기다려요
2. 영희가 와서 뒤에 서요
3. 민수가 와서 제일 뒤에 서요

누가 먼저 탈까요?
→ 철수 → 영희 → 민수 (온 순서대로!)
```

장점: **공폸해 보임** (온 순서대로)
단점: **늦게 온 친구는 오래 기다려야 함**

---

**방법 2: 급한 순서 (Priority)**

```
병원 응급실:
1. 배가 아픈 친구 → 보통 급함
2. 다친 친구 → 급함!
3. 감기 걸린 친구 → 별로 안 급함

누구를 먼저 도와줄까요?
→ 다친 친구 → 배 아픈 친구 → 감기 친구 (급한 순!)
```

장점: **급한 친구 먼저 도와줌**
단점: **안 급한 친구는 너무 오래 기다려요**

그래서 **Aging**이라는 특별한 규칙이 있어요:
- 너무 오래 기다린 친구는 **순위를 높여줌**
- 기다린 시간만큼 더 급해져요!

---

**방법 3: 여러 줄 (MLFQ)**

```
놀이터에 미끄럼틀이 4개 있어요:

빠른 줄 (Q0):     1초 만에 탈 수 있어요
중간 줄 (Q1):     3초 동안 탈 수 있어요
느린 줄 (Q2):     8초 동안 탈 수 있어요
아주 느린 줄 (Q3): 시간 제한 없어요

친구들이 줄을 옮겨요:
- 빠르게 타면 → 그 줄에 계속
- 너무 오래 타면 → 다음 느린 줄로 이동
- 1분마다 → 모두 다시 빠른 줄로!
```

장점: **빠른 친구는 빨리, 느린 친구도 확실히**
단점: **복잡해요**

---

**방법 4: 공평하게 (WFQ)**

```
가족이 피자를 먹어요:

아빠:   100g (성인이니까 많이)
엄마:   100g (성인이니까 많이)
나:     50g  (아이라서 절반)
동생:   10g  (많이 먹으면 아파요)

하지만! 모두가 **공평하게** 먹어야 해요:
- 아빠는 양이 많지만 더 자주 먹지 않아요
- 동생은 양이 적지만 자주 먹을 수 있어요

가상 시간 (Virtual Time)으로 계산해요!
```

장점: **모두에게 공평한 기회**
단점: **계산이 복잡해요**

---

## Part 3: 왜 이것이 필요한가?

### Q: 왜 이런 시스템이 필요한가요?

**A:**

로봇 친구(AI)가 너무 인기가 많아졌어요!

**문제:**
- 친구들이 동시에 로봇한테 도와달라고 해요
- 로봇이 한 번에 한 명만 도와줄 수 있어요
- 어떤 친구는 너무 오래 기다려요

**해결:**
- 똑똑한 순서를 정해요
- 급한 친구 먼저 도와줘요
- 모두를 공평하게 도와줘요

**실제 예:**
- **긴급 메시지:** "도와주세요! 비상사태예요!" → 제일 먼저
- **일반 채팅:** "안녕?" → 나중에
- **대량 작업:** "1000장의 그림 분석해줘" → 제일 나중에

---

## Part 4: 어떻게 작동하나요?

### Q: 컴퓨터가 어떻게 알까요?

**A:**

**준비물:**
1. **Queue (줄 서는 곳):** 기다리는 친구들 목록
2. **Worker (도와주는 사람):** 일 처리하는 로봇
3. **Scheduler (순서 정하는 사람):** 누구를 먼저 도와줄지 결정

**단계:**

```
1. 친구가 요청해요
   ↓
2. Scheduler가 줄에 세워요 (Queue)
   ↓
3. Worker가 친구를 하나 골라요
   ↓
4. 로봇이 일을 해요 (LLM)
   ↓
5. 결과를 친구에게 주어요
```

**비유:**
- **Queue:** 주문 대기표
- **Worker:** 요리사
- **Scheduler:** 주문 받는 직원
- **LLM:** 요리사가 만드는 음식

## Part 5: 재미있는 비유

### 비유 1: 식당 주문

```
FCFS (선착순):
- 먼저 온 손님 주문부터 처리
- 단순하지만, VIP도 기다려야 해요

Priority (급한 순서):
- VIP 손님 먼저 응대
- 하지만 일반 손님은 너무 오래 기다려요
- → Aging: 너무 오래 기다리면 VIP로 승격!

MLFQ (여러 구역):
- 1인 테이블: 빨리 먹고 나가요
- 4인 테이블: 천천히 식사해요
- 룸룸: 시간 제한 없어요
- 1시간마다: 모두 1인 테이블로 이동!

WFQ (공정하게):
- 가족마다 식사 속도가 다르지만
- 모두가 공평하게 밥을 먹어요
- 아빠는 많이, 동생은 조금지만
- 돌아가며 먹어요
```

---

### 비유 2: 학교 수업

```
FCFS:
- 질문한 순서대로 대답해줘요
- 하지만 긴급한 질문도 기다려야 해요

Priority:
- 손을 높이 든 학생 먼저
- 하지만 조용한 학생은 영원히 기다릴 수 있어요
- → Aging: 조용히 기다린 학생도 자리를 줘요

MLFQ:
- 쉬운 질문: 빨리 대답하고 자리로
- 어려운 질문: 천천히 생각해요
- 10분마다: 모두 다시 쉬운 질문부터!

WFQ:
- 반마다 공부하는 속도가 달라요
- 하지만 모두에게 공평한 기회를 줘요
- 빨리하는 반은 자주 짧게
- 천천히 하는 반은 드물게 길게
```

---

### 비유 3: 게임 큐브

```

FCFS:
- 줄 서서 차례대로 타요
- 단순하지만 놀이기구 타는 시간이 달라요
  - 철수: 10분짜리 놀이기구
  - 영희: 1분짜리 놀이기구
  - 영희가 10분이나 기다려요 😞

Priority:
- VIP 패스가 있는 친구 먼저
- 긴급한 친구 먼저
- 하지만 일반 친구는 오래 기다려요
- → Aging: 너무 오래 기다린 일반 친구 VIP로!

MLFQ:
- 빠른 놀이기구 (1분): 재빨리 타고 내려요
  - 짧은 놀이 기구를 좋아하는 친구들
- 느린 놀이기구 (10분): 천천히 즐겨요
  - 긴 놀이 기구를 좋아하는 친구들
- 1시간마다: 모두 다시 빠른 놀이기구부터!

WFQ:
- 친구들마다 티켓 개수가 달라요
  - VIP: 100장
  - 일반: 10장
  - 무료: 1장
- 하지만 모두가 공평하게 놀이기구를 타요
  - VIP는 자주 짧게
  - 무료는 드물게 길게
```

## Part 6: 핵심 개념 쉽게 설명

### Aging (에이징)

**설명:**

오래 기다린 친구를 **더 챙겨주는 것**

```
처음: 보통 친구
기다림: 10초 → 여전히 보통
기다림: 20초 → 조금 급해짐
기다림: 30초 → 급해짐!
기다림: 40초 → 매우 급함!
기다림: 50초 → 최고 긴급! (이제 처리되어요)
```

---

### Boosting (부스팅)

**설명:**

모두를 **다시 공평하게** 시작하는 것

```
처음: 모두가 빠른 줄에
시간 지남: 느린 친구들은 느린 줄에
부스팅: "다 같이 빠른 줄로!"
```

---

### Virtual Time (가상 시간)

**설명:**

**공정한 시간**을 계산하는 마법시계

```
실제 시간: 10분 지났어요
- 텐넌트 A (가중치 100): 가상 시간 0.1초 지남
- 템넌트 B (가중치 10):  가상 시간 1초 지남

왜?
- 템넌트 A는 많이 기여해서 천천히 시간이 가요
- 템넌트 B는 적게 기여해서 빨리 시간이 가요

결과: 모두가 공평하게 일해요!
```

---

## Part 7: 요약

### 핵심 메시지

**이 프로젝트는:**

1. **똑똑한 순서**를 정해요
   - 급한 친구 먼저
   - 오래 기다린 친구 챙겨줘요
   - 모두에게 공평한 기회

2. **4가지 방법**으로 해결해요
   - FCFS: 온 순서대로
   - Priority: 급한 순서대로
   - MLFQ: 빠른/느린 친구 구분
   - WFQ: 공정하게 나눠요

3. **실제로 도움이 돼요**
   - AI(로봇)가 똑똑하게 일해요
   - 모두를 기다리게 하지 않아요
   - 공정하게 도와줘요

---

### 기억할 점

**1. 스케줄링 = 순서 정하기**
- 누구를 먼저 도와줄까요?

**2. 4가지 방법**
- 각각 장단점이 있어요
- 상황에 따라 골라 써요

**3. 공정성이 중요해요**
- 급한 친구를 도와주세요
- 하지만 다른 친구도 잊지 마세요!
- 모두에게 공평한 기회를 주세요
