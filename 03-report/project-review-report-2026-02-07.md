## 6. 발견된 문제점 및 권장사항

### 6.1 사소한 문제점 (미관)

1. **프로젝트 제목 미차이**
   - 문서마다 약간씩 다른 표현 사용
   - **영향**: 없음 (핵심 내용 동일)
   - **권장**: 하나로 통일 (가장 짧은 것 추천)

2. **소수점 표기 차이**
   - 일부 문서에서 반올림 처리
   - **영향**: 없음 (의미 있는 차이 아님)
   - **권장**: 원본 값 유지 또는 일관된 반올림 규칙 적용

### 6.2 개선 권장사항

1. **발표자료 시각 자료**
   - 현재 텍스트 기반 그래프
   - **권장**: 실제 차트/그래프 이미지 추가

2. **MLFQ 향후 실험 언급**
   - 현재 실험에서 FCFS와 동일한 결과
   - **권장**: 장기 작업 포함 실험 추가로 MLFQ 효과 입증
