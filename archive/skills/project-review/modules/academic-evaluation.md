# Academic Evaluation Module

학술적 가치 평가 모듈 - 연구의 학술적 질을 평가합니다.

## 평가 차원: 학술적 가치 (20점)

### 1. 연구 질의의 질 (5점)

**평가 항목:**
- **명확성 (Clarity):** RQ가 구체적이고 명확한가?
  - 모호함 없이 무엇을 연구할지 명시되었는가?
  - 연구의 범위가 명확하게 정의되었는가?

- **실질성 (Substance):** RQ가 실질적인 문제를 다루는가?
  - 실제로 중요한 문제인가?
  - 연구할 가치가 있는 주제인가?

- **실현 가능성 (Feasibility):** RQ가 실현 가능한 범위인가?
  - 주어진 시간과 자원으로 달성 가능한가?
  - 졸업 프로젝트의 규모에 적합한가?

**AI 분석 프롬프트:**
```
다음 논문에서 연구 질의(RQ)를 추출하고 평가하세요:

1. RQ 식별: 명시적/암시적 RQ를 모두 추출하세요.
2. 각 RQ에 대해 다음을 평가하세요 (1-5점):
   - 명확성: 구체적이고 명확한가?
   - 실질성: 실질적인 문제를 다루는가?
   - 실현 가능성: 주어진 범위에서 가능한가?

3. 종합 평가를 제시하세요.
```

### 2. 문헌 조사의 충실성 (5점)

**평가 항목:**
- **최신성 (Recency):** 최신 연구를 포함하는가?
  - 최근 3-5년 내 연구를 포함하는가?
  - 최신 트렌드를 반영하는가?

- **포괄성 (Comprehensiveness):** 관련 분야의 주요 연구를 망라하는가?
  - 해당 분야의 핵심 논문을 포함하는가?
  - 다양한 관점을 고려하는가?

- **비교 분석 (Comparative Analysis):** 단순 나열이 아닌 비교 분석이 있는가?
  - 연구 간의 비교가 있는가?
  - 각 연구의 장단점 분석이 있는가?

- **간극 식별 (Gap Identification):** 연구 간극(gap)을 식별하는가?
  - 기존 연구의 한계를 파악하는가?
  - 본 연구의 필요성을 논리적으로 전개하는가?

**AI 분석 프롬프트:**
```
다음 문헌 조사 섹션을 분석하여 평가하세요:

1. 포함된 논문의 연도 분포를 확인하세요.
2. 주요 연구가 포함되었는지 확인하세요.
3. 비교 분석이 존재하는지 확인하세요.
4. 연구 간극 식별이 있는지 확인하세요.

각 항목에 1-5점을 부과하고 근거를 제시하세요.
```

### 3. 방법론의 타당성 (5점)

**평가 항목:**
- **근거 명시 (Justification):** 선택한 방법의 근거가 명확한가?
  - 왜 해당 방법을 선택했는지 설명하는가?
  - 방법 선택의 타당성을 논증하는가?

- **대안 고려 (Alternative Consideration):** 대안 방법과의 비교가 있는가?
  - 다른 가능한 방법을 고려하였는가?
  - 왜 대안을 선택하지 않았는지 설명하는가?

- **한계 인식 (Limitation Awareness):** 방법의 한계를 인지하는가?
  - 선택한 방법의 한계를 인식하는가?
  - 한계를 보완하는 방안을 제시하는가?

- **적절성 (Appropriateness):** 연구 목적에 부합하는가?
  - 연구 질의에 답하는 데 적절한 방법인가?
  - 데이터와 자원의 효율적 활용인가?

**AI 분석 프롬프트:**
```
다음 방법론 섹션을 분석하여 평가하세요:

1. 선택한 방법과 그 근거를 식별하세요.
2. 대안 방법에 대한 언급이 있는지 확인하세요.
3. 방법의 한계에 대한 언급이 있는지 확인하세요.
4. 연구 목적과의 부합성을 평가하세요.

각 항목에 1-5점을 부과하고 근거를 제시하세요.
```

### 4. 결과 해석 능력 (5점)

**평가 항목:**
- **인사이트 도출 (Insight Generation):** 단순 수치 나열을 넘어 인사이트를 도출하는가?
  - 결과의 의미를 해석하는가?
  - 발견한 사실을 설명하는가?

- **예상치못한 결과 분석 (Unexpected Analysis):** 예상치못한 결과에 대한 분석이 있는가?
  - 예상과 다른 결과를 설명하는가?
  - 그 원인을 추론하는가?

- **RQ와의 연결 (RQ Connection):** 결과가 연구 질의와 잘 연결되는가?
  - 결과가 RQ에 답하는가?
  - 결론이 RQ와 일관되는가?

**AI 분석 프롬프트:**
```
다음 결과 및 결론 섹션을 분석하여 평가하세요:

1. 결과 단순 나열을 넘어 인사이트가 있는지 확인하세요.
2. 예상치못한 결과에 대한 분석이 있는지 확인하세요.
3. 결과가 연구 질의와 연결되는지 확인하세요.

각 항목에 1-5점을 부과하고 근거를 제시하세요.
```
