# 리허설 녹음 자가 평가 폼

> 이 문서의 목적: 리허설 때 스마트폰으로 **자신의 발표를 녹음**하고, 이 폼으로 **객관적으로 평가**한다.
> 사용 방법: 1주 전 1회, 3일 전 2회, 1일 전 1회 — **총 4회 녹음**하고 폼을 채운다. 녹음 파일은 개인 폴더에 보관 (발표 후 삭제 가능).

---

## 🎤 녹음 준비

### 녹음 환경
- [ ] 조용한 방 (TV·음악 OFF)
- [ ] 스마트폰을 평평한 곳에 놓기
- [ ] 스마트폰과 입 사이 거리 **30~50cm** (너무 가까우면 숨 소리 녹음됨)
- [ ] 에어컨·선풍기 소리가 없는 환경

### 녹음 앱
- **iPhone**: 음성 메모 (기본 앱)
- **Android**: Voice Recorder / Otter.ai (선택)

### 녹음 설정
- 파일명: `rehearsal-YYYY-MM-DD-NN.m4a` 또는 `.mp3`
  - 예: `rehearsal-2026-05-19-1.m4a` (1주 전 1차)

---

## 📋 녹음 후 평가 폼

각 리허설마다 아래 폼을 채운다. 체크박스와 점수 (1~5)로 기록.

## 📝 리허설 1차 (발표 1주 전, 5월 19일경)

### 기본 정보
- 녹음 날짜: ______________
- 녹음 시간: ______ 분 ______ 초
- 목표 시간: **슬라이드 1~11 = 10분 20초** / 슬라이드 12 = 30초 / 데모 3분 = 총 13분 50초

### A. 시간 관리

- [ ] 전체 녹음 시간이 **10~14분** 범위인가
- [ ] 슬라이드 1~3 (도입): **2~3분**에 마쳤는가
- [ ] 슬라이드 4~6 (설계): **3~4분**
- [ ] 슬라이드 7~10 (실험): **3~4분**
- [ ] 슬라이드 11~12 (결론·데모): **2~3분**

**점수**: _____ / 5

### B. 말의 속도

- [ ] 한 문장이 **8초 이내**에 말해지는가
- [ ] 긴 문장에서 **호흡을 나눠 쓰는가**
- [ ] 빠른 부분과 느린 부분이 있는가 (강약 조절)
- [ ] 긴장하면 빨라지는 구간을 확인했는가

**점수**: _____ / 5

### C. 발음·목소리

- [ ] 숫자 발음이 또렷한가 (예: "만 일천 팔백사십육" vs "만일천팔백사십육")
- [ ] 전문 용어 발음이 자연스러운가 (FCFS, MLFQ, WFQ, VFT, JFI)
- [ ] 목소리 크기가 일정한가 (중간에 작아지지 않는가)
- [ ] "어..", "음.." 같은 **fill word가 3회 이하**인가

**점수**: _____ / 5

### D. 내용 정확성

- [ ] 핵심 숫자 5개 (약 12배, 약 73%, JFI 1.000/0.316, Rate Limiter 100/50/10/5, gemma4:e4b)를 **모두 정확히** 말했는가
- [ ] 연구 질문 3개 (RQ1/RQ2/RQ3)를 **모두** 언급했는가
- [ ] MLFQ = FCFS인 이유를 **명확히 설명**했는가
- [ ] WFQ의 0.316이 "의도적 차등"임을 **설명**했는가

**점수**: _____ / 5

## 🎯 녹음 듣기 가이드 (비평하는 방법)

### 1회 재생 — 전체 인상

- 말투가 자연스러운가
- 청중 입장에서 따라오기 쉬운가
- 긴장이 느껴지는가

### 2회 재생 — 세부 체크 (위 폼 채움)

- 항목별 체크박스 → 점수
- 막힌 구간 타임스탬프 기록

### 3회 재생 — 개선 시뮬레이션

- 막힌 구간에서 **어떻게 말하면 좋았을지** 다시 해보기
- 개선 버전 다시 녹음 (한 슬라이드만 반복 녹음 OK)

## 🚫 자가 평가 금지 사항

다음은 평가 대상 **아님**:

- 목소리 톤이 높거나 낮음 (본인 목소리이므로 바꾸기 어려움)
- 발표 자세 (녹음으로는 확인 불가)
- 슬라이드 전환 타이밍 (영상 녹화가 필요함)

→ 이 세 가지는 **스마트폰 영상 녹화** (셀프 카메라)로 추가 체크 가능.

## 📊 최종 판정 기준

### A등급 (자신감 있게 발표 가능)
- 총점 16점 이상 / 20점
- 막힘 구간 0~1곳
- 핵심 숫자 5개 모두 정확

### B등급 (조금 더 연습 필요)
- 총점 12~15점 / 20점
- 막힘 구간 2~3곳
- 핵심 숫자 1개 정도 실수

### C등급 (추가 리허설 필수)
- 총점 11점 이하 / 20점
- 막힘 구간 4곳 이상
- 핵심 숫자 2개 이상 실수

→ **C등급이면 다음 주까지 매일 1회 반복 리허설**
